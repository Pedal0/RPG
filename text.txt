On a choisi GitHub pour sa popularité et sa simplicité d’utilisation. GitHub propose des fonctionnalités CI intégrées via GitHub Actions, ce qui facilite l’automatisation des tests et des déploiements directement depuis les dépôts. Il offre une version gratuite pour les étudiants, permettant de gérer le code, les révisions et les pipelines CI en un seul endroit, rendant le développement et l’intégration continue plus efficaces.
